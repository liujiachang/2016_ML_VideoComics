### 项目介绍：
在这高速发展的时期中，各种技术在不断进步，许多新功能也在不断浮现。《VideoComics》也是其中之一，可以带给用户多种享受。<br>
本项目是基于对抗神经网络(gan)完成的风格转换。本项目是把所选中的视频中关键帧提取出来，并且将关键帧图像转换成不同风格的图像。
